# Agent Instructions (Codex)

## Project
March Madness Squares web app (Next.js + Postgres + Auth).

## Rules
- Work in small, reviewable PRs.
- Do not refactor unrelated code.
- Always add/adjust tests for new behavior.
- Always run tests and lint before finishing.
- If tests are flaky, document the cause and make them deterministic.

## Commands
- Install: npm ci
- Dev: npm run dev
- Lint: npm run lint
- Typecheck: npm run typecheck
- Unit tests: npm test
- DB migrate: npm run db:migrate
